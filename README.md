# Django-ToDoList

A simple to do list using django, Built for learning Django


## Getting Started

Follow these steps to run the project locally using a virtual environment.

### 1. Clone the repository

```bash
git clone https://gitea.com/blezecon/Django-ToDoList.git
cd Test-Django-Project
```


### 2. Set Up Venv

```bash
python -m venv env
```
(This is for Mac and Linux)
```bash
source ./env/bin/activate 
```
(This is for Windows)
```bash
source ./env/Scripts/activate 
```

### 3. Install Pakage

```bash
pip install django
```

### 4. Run file

```bash
python manage.py runserver
```
